Detailed Description

Array stack.

This file contains the array stack. It is an array that can grow and shrink when needed. Push and pop operations are O(1).

Hasnain Raza

Initialize the array stack.

This function initializes the array stack. Capacity can't be less than DSC_ARRAY_STACK_INITIAL_CAPACITY. If the capacity is less than DSC_ARRAY_STACK_INITIAL_CAPACITY, it is set to DSC_ARRAY_STACK_INITIAL_CAPACITY. If the free_func is NULL, the data is not freed. If allocation fails, the function returns NULL.

Parameters
capacityInitial capacity of the stack, can not be less than DSC_ARRAY_STACK_INITIAL_CAPACITY
free_funcFunction pointer to free the data
Returns
dsc_array_stack_t* Pointer to the array stack, NULL on failure

Push data to the array stack.

This function pushes data to the array stack. If the array stack is full, the stack is resized. The function returns 0 on success and -1 on failure. Failure can occur if the allocation fails. This function is O(1).

int dsc_array_stack_resize ( dsc_array_stack_t *  array_stack,
int  capacity 
)

Resizes the array stack.

This function resizes the array stack. If the new capacity is less than the current size of the stack, the stack is not modified and the function returns -1. If the new capacity is greater than the current capacity, the stack is resized and the function returns 0. If allocation fails, the function returns -1 and the stack is not modified.

Parameters
array_stackPointer to the array stack, can not be NULL
capacityNew capacity of the stack, can not be less than the current size of the stack
Returns
int 0 on success, -1 on failure
