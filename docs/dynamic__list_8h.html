Detailed Description

Dynamic list.

This file contains the dynamic list. It is an array that can grow and shrink when needed. Insert and remove operations are O(n).

Initialize the dynamic list.

This function initializes the dynamic list. Capacity can't be less than DSC_DYNAMIC_LIST_INITIAL_CAPACITY. If the capacity is less than DSC_DYNAMIC_LIST_INITIAL_CAPACITY, it is set to DSC_DYNAMIC_LIST_INITIAL_CAPACITY. If the free_func is NULL, the data is not freed. If allocation fails, the function returns NULL.

Parameters
capacityInitial capacity of the list, can not be less than DSC_DYNAMIC_LIST_INITIAL_CAPACITY
free_funcFunction pointer to free the data
Returns
dsc_dynamic_list_t* Pointer to the dynamic list, NULL on failure

Inserts an element at the specified index.

This function inserts an element at the specified index. If the index is out of bounds, the function returns -1 and the list is not modified. If the index is equal to the size of the list, the element is appended to the list. If the index is less than the size of the list, the element is inserted at the specified index. If the list is full, the list is resized and the function returns 0. If allocation fails, the function returns -1 and the list is not modified. This function is O(n).

Parameters
dynamic_listPointer to the dynamic list, can not be NULL
indexIndex at which the element is to be inserted, must be within the bounds of the list
dataData to be inserted, can not be NULL
Returns
int 0 on success, -1 on failure

Removes the element at the specified index.

This function removes the element at the specified index. If the index is out of bounds, the function returns -1 and the list is not modified. If the index is less than the size of the list, the element is removed from the specified index. If the list is half full, the list is resized and the function returns 0. If the capacity is less than DSC_DYNAMIC_LIST_INITIAL_CAPACITY, the list is not resized to a smaller size. The function is not affected by the failure of allocation. This function is O(n).

int dsc_dynamic_list_resize ( dsc_dynamic_list_t *  dynamic_list,
int  capacity 
)

Resizes the dynamic list.

This function resizes the dynamic list. If the new capacity is less than the current size of the list, the list is not modified and the function returns -1. If the new capacity is greater than the current capacity, the list is resized and the function returns 0. If the new capacity is less than the DSC_DYNAMIC_LIST_INITIAL_CAPACITY, the list is resized to DSC_DYNAMIC_LIST_INITIAL_CAPACITY and the function returns 0. If the allocation fails, the function returns -1 and the list is not modified.

Parameters
dynamic_listPointer to the dynamic list, can not be NULL
capacityNew capacity of the list, can not be less than the current size of the list
Returns
int 0 on success, -1 on failure
