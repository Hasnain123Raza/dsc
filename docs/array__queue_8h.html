Detailed Description

Array queue.

This file contains the array queue. It is an array that can grow and shrink when needed. Enqueue and dequeue operations are O(1).

Enqueues the data to the array queue.

This function enqueues the data to the array queue. If the array queue is full, the queue is resized and the function returns 0. If the allocation fails, the function returns -1 and the queue is not modified. This function is O(1).
